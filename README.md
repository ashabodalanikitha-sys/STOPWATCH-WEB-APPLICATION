# Stopwatch Web Application

A simple and responsive stopwatch application developed using **HTML**, **CSS**, and **JavaScript**. The application allows users to measure elapsed time accurately with an intuitive interface and essential stopwatch controls.

## Features

* ▶️ **Start** the stopwatch
* ⏸️ **Pause** the running timer
* 🔄 **Reset** the stopwatch to zero
* 🏁 **Lap Time Recording** to track intermediate times
* 📱 Responsive and user-friendly design
* 🎨 Clean UI with modern styling

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

The stopwatch tracks time in hours, minutes, and seconds using JavaScript timer functions. Users can start, pause, reset, and record lap times while the application updates the display in real time.
